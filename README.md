# My-vim-setup
A setup repository I created to be able share my setup across machines while keeping track of any changes I made.

## Installation
Run the following code in your terminal:

  $ cd
  $ git clone git@github.com:rickstaa/.vim.git
  $ ln -s -f .vim/.vimrc
